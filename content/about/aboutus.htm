公司介绍

  环信是移动即时通讯能力的云计算 PaaS (Platform as a Service, 平台即服务) 平台服务商。环信将基于移动互联网的即时通讯能力，通过云端开放的 Rest API 和客户端 SDK 包的方式提供给开发者和企业。环信全面支持Android、iOS、Web等多种平台，在流量、电量、长连接、语音、位置、安全等能力做了极致的优化，让移动开发者摆脱繁重的移动IM通讯底层开发，最大限度地缩短产品开发周期，最短的时间内让App拥有移动IM能力。环信的核心研发团队来自Redhat、Nokia、Polycom等国际知名研发中心，在即时通讯技术领域有多年的深厚积累，更是极客精神和开源软件的积极践行者。环信将以最开放的技术为全球开发者提供最专业最好用的即时通讯云服务；

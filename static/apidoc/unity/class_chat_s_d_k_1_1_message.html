参数
username消息接收者的用户 ID 或群组 ID。
localPath图片的本地路径。

original是否发送原图。
  • true: 发送原图。
  • (默认） false: 发送缩略图。若图片超过 100 KB，SDK 会先压缩图片，然后发送缩略图。 对于 Windows、Unity Mac 或 Unity Windows，SDK 暂不支持压缩功能，只支持原图发送。
width图片宽度，单位为像素。
heigh图片高度，单位为像素。

消息的状态，包含以下状态：

  • CREATE：消息已创建；

  • SUCCESS：消息成功发送；
  • FAIL：消息发送失败。

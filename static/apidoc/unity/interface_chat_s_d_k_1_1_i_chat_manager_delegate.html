◆ OnConversationRead()

void ChatSDK.IChatManagerDelegate.OnConversationRead ( string  from,
string  to 
)

收到会话已读回调。

回调此方法的场景：

  • 消息被接收方阅读（发送了会话已读回执）。

SDK 在接收到此事件时，会将本地数据库中该会话中消息的 isAcked 属性置为 true。

  • 多端多设备登录场景下，一端发送会话已读回执，服务器端会将会话的未读消息数置为 0，

同时其他端会回调此方法，并将本地数据库中该会话中消息的 isRead 属性置为 true。

参数
from已读回执的发送方。
to已读回执的接收方。

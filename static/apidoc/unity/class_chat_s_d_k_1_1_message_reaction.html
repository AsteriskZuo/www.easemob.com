详细描述

消息 Reaction 实体类，有如下属性：

Reaction：消息 Reaction。 UserCount：添加了指定 Reaction 的用户数量。 UserList：添加了指定 Reaction 的用户列表。 State 当前用户是否添加了该 Reaction。

类成员变量说明

◆ Rection

string ChatSDK.MessageReaction.Rection

获取 Reaction 内容。

◆ Count

int ChatSDK.MessageReaction.Count

获取添加了指定 Reaction 的用户数量。

◆ UserList

List<string> ChatSDK.MessageReaction.UserList

获取添加了指定 Reaction 的用户列表。

Note 只有通过 IChatManager#GetReactionDetail 接口获取的是全部用户的分页数据；其他相关接口如IChatManager#GetReactionList等都只包含前三个用户。

返回
用户列表。

◆ State

bool ChatSDK.MessageReaction.State

获取当前用户是否添加过该 Reaction。

  • true：是；
  • false：否。

◆ OnMuteListAddedFromGroup()

void ChatSDK.IGroupManagerDelegate.OnMuteListAddedFromGroup ( string  groupId,
List< string >  mutes,
int  muteExpire 
)

有成员被禁言。

注意

用户禁言后，将无法在群中发送消息，但可查看群组中的消息，而黑名单中的用户无法查看和发送群组消息。

参数
groupId群组 ID。
mutes禁言的成员列表。 Map.entry.key 是禁言成员 ID，Map.entry.value 是禁言时长，单位为毫秒。

How does this class work?

In theory, this class is very straight-forward. It provides a way for multiple delegates to be called, each on its own delegate queue.

In other words, any delegate method call to this class will get forwarded (dispatch_async'd) to each added delegate.

Important note concerning thread-safety:

This class is designed to be used from within a single dispatch queue. In other words, it is NOT thread-safe, and should only be used from within the external dedicated dispatch_queue.

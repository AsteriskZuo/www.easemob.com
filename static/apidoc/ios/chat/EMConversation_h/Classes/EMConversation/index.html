loadNumbersOfMessages:before:

根据时间加载指定条数的消息 

- (NSArray *)loadNumbersOfMessages:(NSUInteger)aCount before:(long long)timestamp;
Parameters
aCount

要加载的消息条数

timestamp

时间点, UTC时间, 以毫秒为单位

Return Value

加载的消息列表

Discussion

  1. 加载后的消息按照升序排列;

  2. NSDate返回的timeInterval是以秒为单位的, 如果使用NSDate, 比如 timeIntervalSince1970 方法，需要将 timeInterval 乘以1000

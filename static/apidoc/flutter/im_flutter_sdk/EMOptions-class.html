EMOptions

EMOptions class

提供 SDK 聊天相关的设置。 用户可以用来配置 SDK 的各种参数、选项， 比如，发送消息加密，是否自动接受加好友邀请。

设置 SDK Param appKey 创建 app 时在 console 后台上注册的 app 唯一识别符。

startTs property Null safety

int startTs
final

消息查询的起始时间，Unix 时间戳，单位为毫秒。默认为 -1，表示消息查询时会忽略该参数。 若 startTs 设置为特定时间点，而 endTs 采用默认值 -1，则查询起始时间至当前时间的消息。 若 startTs 采用默认值 -1，而 endTs 设置了特定时间，SDK 返回从会话中最早的消息到结束时间点的消息。
